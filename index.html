FindArtists

FindArtists is a python app to find and gather information on well-known musicians and recording artists.

The Problem

The idea to find information on musicians came when Charlie Marshall needed to conduct market research into the electronic industry. In particular, information on popular artists within the industry. Most data-vendors were unreliable, did not hold required information that was being looked for or were far too expensive. Searching for this data manually would've been a slow, tedious process, given the necessary number of artists to find information for.

Purpose

FindArtists is an application which is being developed to solve or otherwise mitigage the issues defined in the The Problem section above. As such, it can be surmised that FindArtists must fulfill several goals in order to be a for its intended purpose:

  • FindArtists must provide be cheap and/or free for the end-user. This will solve the issue of high cost typically associated with finding such information from data vendors.
  • FindArtists must run quickly. This will solve the issue the user has in self-attaining the data.
  • FindArtists must not require too much human intervention. This will minimise costs and manpower for the end user, which is typically associated with manual searching for this data.
  • FindArtists must output information which is conducive to market research.
  • FindArtists must be able output a large quantity of relevant information.
  • FindArtists must output accurate and valid information. If it does not fulfill this, it cannot be used by the end-user correctly and therefore will not achieve its intended purpose.

Features/Specifications

In order to fulfil the intended purpose and resolve the problems defined, FindArtists must meet a series of specifications:

  • Take user inputs within a .csv, .xlsx OR a user-input textbox.
  • Output data in a user-defined .csv OR .xlsx format.
  • Allow the user to define an output filename.
  • Allow the user to define an output filepath.
  • Allow the user to define an input filename.
  • Allow the user to define an input filepath.
  • Have an easy-to-use intuitive UI, to ensure the end-user doesn't require much technical knowledge.
  • Allow the user to input a large volume of artists names.
  • Allow the user to output only their desired fields.
  • Inform the user of the progress of the apps execution.

Intended Audience

  • This project is for Entrepreneurs and Market Researchers who are looking for a way to easily find and download recording artist information.
  • No prior experience with programming is required.
  • This project can be used to find information about artists, such as their name, genre, and net worth.
  • FindArtists can find, gather and return information on a large quantity or artists in a far shorter time frame, with less mistakes, than it would take a human.
